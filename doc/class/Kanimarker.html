Class: Kanimarker

Defined in: kanimarker.coffee

Overview

OpenLayers 3 でマーカーを表示するクラス

Examples:

マーカーを 表示/非表示 する 

kanimarker = new Kanimarker(map)
kanimarker.setPosition(point, accuracy = 50)
kanimarker.setPosition(null)

Author:

  • sakai@calil.jp

Version:

  • 1.2

# (void) setPosition(toPosition, accuracy, silent = false)

現在地マーカーを移動する

Parameters:

  • toPosition ( Array ) — 現在地を描画する座標
  • accuracy ( Number ) — 円の広さ (メートル) 指定がない場合は変更されない
  • silent ( Boolean ) — renderをしない時に true にする. default: false
